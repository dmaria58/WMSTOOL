输入多个数据

## 何时使用

- 输入多个数据,多个值是逗号分隔的字符串"aa,bb,ccc"
- 鼠标移入下拉展示当前数据

## API

### MultipleInput

| Property | Description | Type | Default Value |
| --- | --- | --- | --- |
| value | 当前操作数据| string |""|
| onChange | 回调时间 | func | 无 |
