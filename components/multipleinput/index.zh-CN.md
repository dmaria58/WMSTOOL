## API

### MultipleInput

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 当前操作数据| string | 无 |
| dropDown | 是否展开下拉框数据 | boolean | true |
| valueData | 数据| object[] | [] |
| onChange | 回调时间 | func | 无 |
