查询框折叠框。

## 何时使用

- 用于展开折叠

## API

### Easyform

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| rules | 验证规则 maxnum(最大),minnum（最小）,pattern（正则），required(必填)| object[] | [] |
| easyCheckValue | 需要验证的value | string | string |
| easyCheck | 非修改input框开启验证（如最后submit，但是输入框未做修改） | boolean | false |
| isright | 验证成功与否参数返回 | boolean |  |
